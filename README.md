# timer
A small command line sport timer 

## Dependencies

`mpv`

## Usage

| action | key |
|--|--|
| pause / unpause | space |
| next activity | `n` or ENTER |
| previous activity | `p` |
| go 5 secs forward | `f` |
| go 5 secs backward | `b` |
| restart activity | `r` |
| quit tool | `q` |

if you are in pause-mode you have to unpause to able to use any other actions.
